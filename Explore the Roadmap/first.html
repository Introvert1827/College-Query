First Year Roadmap

Here are some pro tips that you must follow in first year inorder to get good result in future

Note :-Follow the below procedure only if you want to get into the software field.

Some of the below step might confuse you if you heard this first time , but don't get stuck on it by time of learning you will familiar with this.

  • Learn any programming language either C++ or Java. I never prefer you to start with python (even though it is very beginner friendly) because you get more conceptual clarity in learning C++ and Java.

  • Now if you are familiar with the any programming language and know some basic syntax, it's is time to learn some Data Structure and Algorithm ( I am not saying all only some).

  • Go to Geeks for Geeks portal click on Data Structure section and start with linked list. I prefer you learn from youtube videos then read article of geeksforgeeks and practice on them.

  • Since it may take some time so be patience and don't give up after 2 or 3 months of practice.

  • During learning data structure you might find somewhere the word "Algorithm" for example by using "Two Pointer Algorithm" so when you learning data structure you simultaneaously learning some algorithm as well.

  • Since learning data structure and algorithm and solving problem on them is never ending process and it is improves by day to day practice.

  • Practice problem and participate in contest from different website like :
    • HackerEarth
    • HackerRank
    • CodeChef
    • Codeforces
    • leetcode
    • Interview bit

  • By solving problem and participating in contest help you to increase your problem solving skill which is a prequsite as a software enginner

Note :
  • You face some time constraints here because there are lot other things that you have to do in your freshman year like understanding college syllabus and exam pattern, making friends, scholarship form etc.

  • So try to give some fixed 2-3 hours on daily basis.

  • Give more time on sundays and winter and summer break.

  • If you come in 2nd year and not able complete data structure algorithm then no need to worry about that, as I said its never ending process and you have to practice it on daily basis.You can move to 2nd year roadmap as you come in 2nd year.
